# Carbon Compass

Carbon Compass is a personal carbon-footprint navigator that helps individuals understand, track, and reduce their emissions through small weekly actions and personalized insights.

## Solution

The prototype turns carbon tracking into a simple route:

1. Enter weekly habits for home energy, transport, food, shopping, and waste.
2. See an instant footprint estimate in kg CO2e.
3. Use the compass to identify the largest source.
4. Plan simple actions with estimated weekly savings.
5. Chat with EcoGuide for a first move, score explanation, or 7 day plan.
6. Save each week and watch the footprint trend over time.

## What Makes It Unique

Instead of showing only a score, Carbon Compass shows a direction. The compass points to the category that currently needs attention, then the route planner recommends the next best action based on the user's own inputs.

The redesigned prototype uses a dark animated command-center interface, live scenario controls, an impact pulse meter, and a local chatbot that reads the user's current calculator state.

## Key Features

- Understand: plain explanations of carbon footprint, big emission levers, and why weekly tracking works.
- Track: a weekly calculator, saved footprint log, and visual trend chart.
- Reduce: action cards with effort level, category, and estimated kg CO2e savings.
- Personalize: recommendations change based on the user's largest source, travel habits, diet, renewable energy use, and goal level.
- Simple actions: low-friction steps such as commute swaps, appliance scheduling, low-carbon meals, reuse stations, and purchase delays.
- Engagement: animated compass, live challenge prompts, scenario demos, hover feedback, and chatbot prompt chips.
- Chatbot: EcoGuide answers common user questions using the current footprint, top category, planned savings, and recommended actions.

## Prototype Files

- `index.html`: app structure and content.
- `styles.css`: responsive visual design.
- `app.js`: calculator, insights, action planner, saved history, and chart logic.

Open `index.html` in a browser to use the prototype.
